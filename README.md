## ğŸš€ Getting Started

### Prerequisites

- Java Development Kit (JDK) 21 or higher
- Gradle 8.5 or higher
- JavaFX 21

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Arinonia/altiscore-launcher.git
```

2. Navigate to the project directory:
```bash
cd altiscore-launcher
```

3. Build the project:
```bash
gradle build
```

4. Run the launcher:
```bash
gradle run
```
